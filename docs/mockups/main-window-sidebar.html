Left: the label sidebar (color swatch + name + count) replacing the collection tab bar, with All + Best pinned and an OR/AND combine toggle. Top: the toolbar — a name filter (replaces the separate Ctrl+F search dialog), an inline frames-per-preview picker (card density), and a single Sort button whose face shows the current field + direction and opens the popover (field, direction, favorites-first), drawn open here. Each card's footer is a single row filling the former empty caption space — the Best-star toggle (filled/gold when in Best, faint outline otherwise), then the label dots, then the name filling the rest. The highlighted card with the floating night pill shows the drag-a-label-onto-cards assignment gesture. Sketch, not pixel-final.
